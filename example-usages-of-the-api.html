Example Usages Of The API

Float Data Example

  • Let's assume a scenario which you want to save a float data named PlayerMoney. After that you want to load the data and then delete it. Following example is the guide on just how to do it.

Saving Float Data

const FString FilePath = USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN); const FString PlayerMoneyString = "PlayerMoney"; USaveLoadManager::SaveData(PlayerMoneyString, USaveLoadManager::FloatToByteArray(PlayerMoney), EDataType::FloatType, FilePath);

Loading Float Data

EDataType DataType; TArray<uint8> ByteArrayPlayerMoney; const FString FilePath = USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN); const FString PlayerMoneyString = "PlayerMoney"; if (USaveLoadManager::LoadData(PlayerMoneyString, ByteArrayPlayerMoney, DataType, FilePath)) { if (DataType == EDataType::FloatType) { PlayerMoney = USaveLoadManager::ByteArrayToFloat(ByteArrayPlayerMoney); UE_LOG(LogTemp, Log, TEXT("Loaded PlayerMoney: %f"), PlayerMoney); } else { UE_LOG(LogTemp, Warning, TEXT("Unexpected data type for PlayerMoney")); } } else { UE_LOG(LogTemp, Error, TEXT("Failed to load data for key '%s'"), *PlayerMoneyString); }

Deleting Float Data

const FString FilePath = USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN); const FString PlayerMoneyString = "PlayerMoney"; USaveLoadManager::DeleteData(PlayerMoneyString, FilePath);

Tip: Unlike saving and loading, you do not need to specify the data type when deleting data.

More Specific Use Cases

  • These are some of the more specific use cases which you might need to use in your project.

Bulk Saving, Loading and Deleting Example

  • Let's assume you want to save and load an enum variable and a transform variable for each Table actor with unique IDs in your level. Following example is the guide on just how to do it.

Bulk Saving Data Example

const FString FilePath = USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN); TArray<AActor*> Tables; UGameplayStatics::GetAllActorsOfClass(GetWorld(), ATable::StaticClass(), Tables); int32 TableNumber = 0; UE_LOG(LogTemp, Warning, TEXT("Starting to save tables...")); for (AActor* Actor : Tables) { if (const ATable* Table = Cast<ATable>(Actor)) { FString UniqueTransformKey = "TableTransform" + FString::FromInt(TableNumber); FString UniqueTableTypeKey = "TableType" + FString::FromInt(TableNumber); USaveLoadManager::SaveData(UniqueTransformKey, USaveLoadManager::TransformToByteArray(Table->GetActorTransform()), EDataType::TransformType, FilePath); USaveLoadManager::SaveData(UniqueTableTypeKey, USaveLoadManager::EnumToByteArray(static_cast<uint8>(Table->TableType)), EDataType::EnumType, FilePath); UE_LOG(LogTemp, Warning, TEXT("Saved table #%d, type=%d, transform=%s"), TableNumber, static_cast<uint8>(Table->TableType), *(Table->GetActorTransform().ToString())); } TableNumber++; } UE_LOG(LogTemp, Warning, TEXT("Finished saving tables."));

Bulk Loading Data Example

if (UWorld* World = GetWorld()) { int32 TableNumber = 0; EDataType DataType; while (true) { FString UniqueTransformKey = "TableTransform" + FString::FromInt(TableNumber); FString UniqueTableTypeKey = "TableType" + FString::FromInt(TableNumber); TArray<uint8> ByteArray; if (USaveLoadManager::LoadData(UniqueTransformKey, ByteArray, DataType, USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN))) { FTransform LoadedTransform = USaveLoadManager::ByteArrayToTransform(ByteArray); if (USaveLoadManager::LoadData(UniqueTableTypeKey, ByteArray, DataType, USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultSaveFileName(), ESaveFileFormat::BIN)) && DataType == EDataType::EnumType && ByteArray.Num() > 0) { switch (static_cast<ETableType>(ByteArray[0])) { case ETableType::Table1: if (ATable* Table = World->SpawnActor<ATable>(BP_Table1, LoadedTransform)) { Table->bIsPlaced = true; } break; case ETableType::Table2: if (ATable* Table = World->SpawnActor<ATable>(BP_Table2, LoadedTransform)) { Table->bIsPlaced = true; } break; default: UE_LOG(LogTemp, Error, TEXT("Unknown table type encountered while initializing tables.")); break; } } TableNumber++; } else { break; } } }

Bulk Deleting Data Example

" Warning: If you are using the bulk saving method (as described above), it is recommended to delete the entire save file before saving again. The reason is that deleting saved data using a for loop is unreliable. If any actor of the saved class is destroyed in the level, the for loop may go out of bounds. This happens because the loop expects a specific number of actors, but when an actor is missing, the loop fails to properly delete that entry. As a result, the saved data will never be fully deleted unless the number of actors of that class in the level matches or exceeds the saved number of actors. To avoid this issue, deleting the file entirely and resaving ensures that the saved data is accurate and up-to-date."

const FString FilePath = USaveLoadManager::PrepareFilePath(USaveLoadManager::GetDefaultFileName, ESaveFileFormat::BIN); USaveLoadManager::DeleteFile(FilePath);

" Tip: You can use a custom save file name instead of the default one to keep different types of data separate. This way, you don't need to clean up and resave all the saved data when you only want to save specific data in bulk."

" Info: The examples in this documentation can be applied to any supported data type. Simply review the relevant sections above and implement the appropriate logic for the data type you need to work with."
