EDataType

  • This enumeration represents the different variable data types available in Unreal Engine. It is used to define the type of data stored in a variable.

Enum Values

  • FloatType: Represents a floating-point number, used for precise calculations.

  • DoubleType: Represents a double precision floating-point number, used for high precision calculations.

  • BoolType: Represents a boolean type, used for true/false conditions.

  • IntType: Represents an integer number, used for whole number calculations.

  • FStringType: Represents a string in Unreal Engine, used for text-based data.

  • EnumType: Represents an enumeration, used to define a type of element that consists of named constants.

  • VectorType: Represents a Vector, which is a structure used to hold a three-dimensional point such as position or direction.

  • RotatorType: Represents a Rotator, which is a structure used to hold rotation in 3-dimensional space.

  • TransformType: Represents a Transform, which is used to store a combination of translation (position), rotation, and scale.
